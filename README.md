# agency-state-brand

The canonical source of truth for how Agency State presents itself. Holds **approved brand truth only** — no drafts, no how-tos, no actors. This repo feeds the Agency State brand MCP, so a workflow fetching from it always gets the approved answer, never a half-formed one.

## Contents

- `brand-platform.md` — AI-facing brand intelligence: identity, what Agency State delivers, positioning, audience, voice, vocabulary, never-say, proof, boundaries. The primary MCP source; each section is written to be served whole as one MCP tool response.
- `brand-guidelines.md` — visual identity: logo, typography, color, components, page types.

## Organizing principle

Everything here answers one question: **"What is true and approved about how Agency State presents itself?"** Finished, canonical statement → belongs here. Draft, experiment, how-to, or actor → lives elsewhere (the `agency-state-practice` repo holds working notes and work-in-progress).

## Provenance

Extracted from `agency-state-practice` on 2026-05-28. `brand-platform.md` supersedes the former brand book (v2.1); the superseded snapshot and prior history live in `agency-state-practice`.
